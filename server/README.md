# Ryan 理发馆后端

## 准备
1. 在 supabase.com 创建项目
2. SQL Editor 执行 `sql/schema.sql`
3. Storage 创建两个 **public** bucket：`avatars`、`store-images`
4. 复制 `.env.example` 为 `.env` 并填入 SUPABASE 三个值
5. `npm install`
6. `npm run seed` 灌入演示数据

## 运行
- `npm start` 启动服务（默认 http://localhost:3000）
- `npm test` 运行测试

## 默认管理员
admin / admin123
